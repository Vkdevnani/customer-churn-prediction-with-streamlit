# Customer Churn Prediction

This project predicts customer churn using machine learning. Built during my internship at Celebal Technologies using the Telco dataset from Kaggle.


## Features

- Full EDA, preprocessing and model training in Jupyter notebook
- Interactive Streamlit app for:
  - Single-customer prediction
  - Batch prediction via CSV
- Downloadable results


## ML models used

- Logistic regression
- Scikit-learn pipeline
- StandardScaler + LabelEncoding
